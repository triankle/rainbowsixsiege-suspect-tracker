How it works

Copy ranked numbers from R6 Tracker. Ranked unlocks at account level 50 in-game; if the level is below that, the ranked stats are not valid for this tool.

Tick every ranked season this account has played. The tool looks at K/D and win rate only after enough ranked games, checks season pattern (only current season vs. long breaks), and separates cheat-like stats from smurf-like history.

Player stats

Must be at least 50 for ranked — the form blocks analysis if lower.
Ranked seasons played

Check each season this account has played ranked. The last one is treated as the current season (change the constant in script.js when Ubisoft releases a new season).
